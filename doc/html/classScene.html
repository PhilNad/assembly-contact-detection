MatrixX3f Scene::get_contact_convex_hull ( string  id1,
string  id2 = "",
int  vertex_limit = 255 
)

Return the vertices of the convex hull over the contact points between two objects, or between an object and the scene.

Parameters
id1ID of the (first) object for which to get the contact points.
id2ID of the second object on which the contact points lie. If empty, use all contact points lying on the first object.
vertex_limitMaximum number of vertices of the convex hull (default: 255)
Returns
Nx3 matrix representing the subset of contact points lying on the convex hull.
Note
A small perturbation is added to each contact point to make sure that the convex hull is full dimensional. Equivalent to QHull's "QJ" option.

◆ get_contact_points()

std::vector< Contact > Scene::get_contact_points ( string  id1,
string  id2 = "",
bool  penetrating = false 
)

Get the contact points between two objects that are not penetrating.

Parameters
id1id of the first object
id2id of the second object (optional)
penetratingIf true, return penetrating contact points. Otherwise, return surface contact points (default).
Returns
vector of contact points between the two objects
Note
If id2 is not set, returns all contact points involving id1.
The normal of a contact point is the one at the surface of object ID1. get_contact_points(id1, id2) =/= get_contact_points(id2, id1).
